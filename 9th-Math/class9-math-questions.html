Chapter 1: Number System

Write the decimal expansion of 1/7. Is it terminating or non-terminating?
1/7 = 0.142857... (repeating). It is non-terminating and recurring, hence rational.
Represent √5 on the number line geometrically.
Construct a right triangle with legs 2 and 1 units. Hypotenuse = √5. Use compass to mark it on the number line.
Rationalize the denominator of 1/(√7 + √6).
Multiply by conjugate: \( \frac{1}{\sqrt{7} + \sqrt{6}} \times \frac{\sqrt{7} - \sqrt{6}}{\sqrt{7} - \sqrt{6}} = \sqrt{7} - \sqrt{6} \)
Find two rational numbers between √2 and √3.
√2 ≈ 1.41, √3 ≈ 1.73. Rational numbers between them: 1.5 and 1.6.
Express 0.333... as a rational number.
Let x = 0.333... ⇒ 10x = 3.333... ⇒ 9x = 3 ⇒ x = 1/3
Is π a rational or irrational number? Justify your answer.
π is irrational. Its decimal expansion is non-terminating and non-repeating: 3.14159...

Chapter 2: Polynomials

Find the degree of the polynomial 5x³ - 2x² + x - 7.
The degree is 3 (highest power of x).
If p(x) = x² - 3x + 2, find p(1), p(2), and p(0).
p(1) = 0, p(2) = 0, p(0) = 2
Factorize x² - 5x + 6.
x² - 5x + 6 = (x - 2)(x - 3)
Show that x = 2 is a zero of the polynomial x² - 4x + 4.
Substitute x = 2: 4 - 8 + 4 = 0 ⇒ x = 2 is a zero.
Divide x³ + 3x² + 3x + 1 by x + 1.
Using long division or identity: (x + 1)³ = x³ + 3x² + 3x + 1

Chapter 3: Linear Equations in Two Variables

Write the standard form of a linear equation in two variables.
Ax + By + C = 0, where A, B, and C are real numbers and A, B ≠ 0.
Find two solutions of the equation 2x + 3y = 6.
x = 0 ⇒ y = 2; x = 3 ⇒ y = 0
Draw the graph of the equation x + y = 4.
Plot points like (0,4), (2,2), (4,0) and draw a straight line through them.
If x = 1, find the value of y in the equation 3x + 2y = 7.
3(1) + 2y = 7 ⇒ 2y = 4 ⇒ y = 2
Write a linear equation whose graph passes through the points (0, 2) and (2, 0).
Slope = (0 - 2)/(2 - 0) = -1 ⇒ Equation: x + y = 2
